# casproxyserver
**casproxyserver** is a cross-platform smartcard proxy server.

## Features
- Supports WinSCard API calls:
  - `SCardEstablishContext`
  - `SCardReleaseContext`
  - `SCardListReaders`
  - `SCardConnect`
  - `SCardDisconnect`
  - `SCardBeginTransaction`
  - `SCardEndTransaction`
  - `SCardTransmit`
  - `SCardGetAttrib`
